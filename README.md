## ✨ Project Overview

Welcome to **Ex-Book Website** — a clean, modern, and responsive platform designed for book lovers to discover and explore a curated collection of books online. This website offers an immersive reading experience with carefully selected books across various genres, providing readers with an intuitive interface to browse, discover, and get inspired by literary works.

Whether you're an avid reader looking for your next favorite book or just beginning your reading journey, Ex-Book Website delivers a seamless, visually appealing experience that works flawlessly across all devices.

## 🚀 How to Run Locally

### Method 1: Simple Setup
1. **Clone or download the repository**
   ```bash
   git clone https://github.com/sushantku1099/Ex-Book_Website.git
   cd Ex-Book_Website
   ```

2. **Open the main file**
   - Navigate to the project directory
   - Double-click on `index.html` to open in your default browser
   - Or right-click and choose "Open with" your preferred browser

### Method 2: Using Live Server (Recommended for Development)
1. **Install Live Server extension** in VS Code
2. **Open the project folder** in VS Code
3. **Right-click on `index.html`** and select "Open with Live Server"
4. The website will automatically open in your browser with live reload capability
